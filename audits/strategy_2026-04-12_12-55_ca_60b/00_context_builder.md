# Context Builder

## System Structure

The provided code is part of a larger system called RUFLO, which is a business-agnostic strategy engine that thinks like a human strategist. The main purpose of the system is to help businesses develop strategies based on their industry and target audience.

The core modules of the RUFLO system are:

1. **Audience Intelligence Engine**: This module uses deep learning algorithms to analyze customer data and identify patterns in customer behavior, preferences, and interests. It also performs intent classification and scoring to determine the most relevant keywords and topics for a given business.
2. **Context Engine**: This module generates unique keyword clusters based on location, religion, and language. It helps businesses understand their target audience better by identifying the most relevant keywords and topics for each customer segment.
3. **DeepSeek (Ollama)**: This module is responsible for audience chain expansion, which involves finding new customers based on their interests and behaviors. It also performs intent classification and scoring to determine the most relevant keywords and topics for a given business.
4. **Gemini (free)**: This module is responsible for persona elaboration, which involves creating detailed customer personas based on their interests, behaviors, and preferences. It also generates context keyword clusters that are specific to each customer segment.
5. **Claude Sonnet**: This module is responsible for final strategy synthesis, which involves combining the insights generated by the other modules to create a comprehensive strategy for a given business.

The data flow in the RUFLO system is as follows:

1. The user inputs the industry and target audience for the business they want to develop a strategy for.
2. The Audience Intelligence Engine analyzes customer data to identify patterns in customer behavior, preferences, and interests. It also performs intent classification and scoring to determine the most relevant keywords and topics for the given business.
3. The Context Engine generates unique keyword clusters based on location, religion, and language. It helps businesses understand their target audience better by identifying the most relevant keywords and topics for each customer segment.
4. The DeepSeek (Ollama) module finds new customers based on their interests and behaviors. It also performs intent classification and scoring to determine the most relevant keywords and topics for the given business.
5. The Gemini (free) module creates detailed customer personas based on their interests, behaviors, and preferences. It also generates context keyword clusters that are specific to each customer segment.
6. The Claude Sonnet module combines the insights generated by the other modules to create a comprehensive strategy for the given business.
